# 欢迎

欢迎使用 **Nextra SaaS 模板**文档！这个模板提供了一个现代化、功能强大的基础，用于使用 Nextra、Next.js 和 Tailwind CSS 构建 SaaS 网站。

## 包含内容

这个模板包含了开始所需的一切：

- **可定制的落地页** - 现代化、响应式设计，包含预构建组件
- **完整的 i18n 支持** - 开箱即支持英语、中文和西班牙语
- **集成文档系统** - 由 Nextra 提供支持，包含搜索和导航
- **深色/浅色主题** - 流畅的主题切换，支持系统偏好设置
- **现代化 UI 组件** - 使用 Radix UI 和 Tailwind CSS 构建

## 快速开始

1. 克隆仓库
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 打开 [http://localhost:3000](http://localhost:3000)

## 你可以构建什么

使用这个模板，你可以构建：

- SaaS 落地页
- 产品文档站点
- 营销网站
- 作品集网站
- 以及更多！
