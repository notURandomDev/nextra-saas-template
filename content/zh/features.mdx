# 功能特性

Nextra SaaS 模板开箱即用，包含许多强大的功能。

## 落地页

### 英雄区域

引人注目的英雄区域，包含：

- 可自定义的徽章
- 标题和副标题
- 行动号召按钮
- 带头像的社交证明

### Logo 跑马灯

动画 Logo 展示，显示：

- 技术合作伙伴
- 工具和框架
- 无限滚动动画

### Bento 网格

现代网格布局，包含：

- 带系统状态的指标卡片
- 带标签的功能亮点
- 交互式图表
- 可自定义的跨度（宽、正常）

### 定价部分

灵活的定价显示，包含：

- 多个定价计划
- 功能列表
- 高亮显示的推荐计划
- 可自定义的 CTA 按钮

### 页脚

全面的页脚，包含：

- 品牌描述
- 链接组
- 社交媒体链接
- 版权声明

## 文档系统

由 Nextra 提供支持，包含：

- **搜索** - 所有页面的全文搜索
- **导航** - 自动生成的目录
- **语法高亮** - 带有适当高亮的代码块
- **LaTeX 支持** - 数学符号支持
- **响应式设计** - 移动友好的布局

## i18n 支持

内置国际化，包含：

- **三种语言** - 英语、中文、西班牙语
- **易于扩展** - 简单的字典系统
- **类型安全** - 完整的 TypeScript 支持
- **基于 URL 的路由** - `/en/*`、`/zh/*`、`/es/*`

### 添加新语言

1. 在 `app/_dictionaries/[lang].ts` 中创建字典文件
2. 将语言添加到 `i18n-config.ts`
3. 在 `get-dictionary.ts` 中添加导入
4. 在 `content/[lang]/` 中创建内容目录

## 主题支持

### 深色/浅色模式

- **系统默认** - 遵循用户的系统偏好
- **手动切换** - 用户可以切换主题
- **流畅过渡** - 动画主题变化
- **持久化** - 主题偏好保存到 localStorage

### 自定义

在 `app/_resources/theme.config.tsx` 中编辑主题颜色：

```typescript
primaryHue: 211,        // 主色相 (0-360)
primarySaturation: 100, // 主色饱和度 (0-100)
```

## UI 组件

使用现代技术构建：

- **Radix UI** - 可访问的无样式组件
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Lucide Icons** - 美观一致的图标
- **Framer Motion** - 流畅的动画

### 可用组件

- 按钮（主要、轮廓、幽灵）
- 卡片和容器
- 导航菜单
- 模态框和对话框
- 表单元素
- 以及更多！

## 性能

为速度优化：

- **静态生成** - 预渲染页面
- **代码分割** - 自动包优化
- **图像优化** - Next.js Image 组件
- **字体优化** - Next.js Font 组件

## 开发体验

为开发者构建：

- **TypeScript** - 完整的类型安全
- **热重载** - 更改时的即时反馈
- **ESLint** - 代码质量检查
- **Prettier** - 代码格式化（可选）
