# Bienvenido

¡Bienvenido a la documentación de la **Plantilla SaaS de Nextra**! Esta plantilla proporciona una base moderna y poderosa para construir sitios web SaaS con Nextra, Next.js y Tailwind CSS.

## Qué Incluye

Esta plantilla incluye todo lo que necesitas para comenzar:

- **Landing Page Personalizable** - Diseño moderno y responsivo con componentes preconstruidos
- **Soporte i18n Completo** - Inglés, chino y español de fábrica
- **Documentación Integrada** - Potenciada por Nextra con búsqueda y navegación
- **Temas Oscuro/Claro** - Cambio suave de temas con soporte de preferencia del sistema
- **Componentes UI Modernos** - Construidos con Radix UI y Tailwind CSS

## Inicio Rápido

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`
4. Abre [http://localhost:3000](http://localhost:3000)

## Qué Puedes Construir

Con esta plantilla, puedes construir:

- Landing pages SaaS
- Sitios de documentación de productos
- Sitios web de marketing
- Portafolios
- ¡Y mucho más!
