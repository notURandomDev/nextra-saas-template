# Welcome

Welcome to the **Nextra SaaS Template** documentation! This template provides a modern, powerful foundation for building SaaS websites with Nextra, Next.js, and Tailwind CSS.

## What's Included

This template comes with everything you need to get started:

- **Customizable Landing Page** - Modern, responsive design with pre-built components
- **Full i18n Support** - English, Chinese, and Spanish out of the box
- **Integrated Documentation** - Powered by Nextra with search and navigation
- **Dark/Light Themes** - Smooth theme switching with system preference support
- **Modern UI Components** - Built with Radix UI and Tailwind CSS

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## What You Can Build

With this template, you can build:

- SaaS landing pages
- Product documentation sites
- Marketing websites
- Portfolio sites
- And much more!
