# Features

Nextra SaaS Template comes with many powerful features out of the box.

## Landing Page

### Hero Section

Eye-catching hero section with:

- Customizable badge
- Headline and subheadline
- Call-to-action buttons
- Social proof with avatars

### Logo Marquee

Animated logo showcase displaying:

- Technology partners
- Tools and frameworks
- Infinite scroll animation

### Bento Grid

Modern grid layout featuring:

- Metrics cards with system status
- Feature highlights with tags
- Interactive charts
- Customizable spans (wide, normal)

### Pricing Section

Flexible pricing display with:

- Multiple pricing plans
- Feature lists
- Highlighted recommended plan
- Customizable CTA buttons

### Footer

Comprehensive footer with:

- Brand description
- Link groups
- Social media links
- Copyright notice

## Documentation System

Powered by Nextra, includes:

- **Search** - Full-text search across all pages
- **Navigation** - Auto-generated table of contents
- **Syntax Highlighting** - Code blocks with proper highlighting
- **LaTeX Support** - Mathematical notation support
- **Responsive Design** - Mobile-friendly layout

## i18n Support

Built-in internationalization with:

- **Three Languages** - English, Chinese, Spanish
- **Easy Extension** - Simple dictionary system
- **Type Safety** - Full TypeScript support
- **URL-based Routing** - `/en/*`, `/zh/*`, `/es/*`

### Adding New Languages

1. Create dictionary file in `app/_dictionaries/[lang].ts`
2. Add language to `i18n-config.ts`
3. Add import in `get-dictionary.ts`
4. Create content directory in `content/[lang]/`

## Theme Support

### Dark/Light Mode

- **System Default** - Follows user's system preference
- **Manual Toggle** - Users can switch themes
- **Smooth Transitions** - Animated theme changes
- **Persistent** - Theme preference saved to localStorage

### Customization

Edit theme colors in `app/_resources/theme.config.tsx`:

```typescript
primaryHue: 211,        // Primary color hue (0-360)
primarySaturation: 100, // Primary color saturation (0-100)
```
